# CodeToAGI — Episode 36 Challenge

**Product Catalog Formatter** using Python String Formatting (f-strings + format spec)

## 🎯 Challenge Requirements
- Format prices as currency with 2 decimal places
- Right-align stock counts in a fixed width
- Clearly flag out-of-stock or low-stock items
- Bonus: Use `Template` for a safe, user-editable receipt

## 🚀 Solution Features
- Clean, professional aligned table output
- Proper currency formatting with commas
- Visual stock status indicators
- Safe receipt template using `string.Template`
